  • 🔴 Двигател (Fuel type) is completely missing from the rendered form — it's in state but never shown. Needs to be added in Row 1 between Модификация and Състояние.
  • 🟡 Price section layout — current form uses nested sub-boxes (price+vat+currency together, month+year together). Mobile.bg has all price/date/mileage fields in one flat row.
  • ⚪ Row 2, 3, location, extras, description — already match mobile.bg order.
