React Page Builder Libraries

What's actually available and maintained in 2025

Craft.js React 19 ✓ v0.2.12
MIT · Open source

Headless drag-and-drop page builder. You define block components, it handles the editor canvas — selection, drag, config panels, undo/redo, serialization to JSON.

✓ Most mature headless option ✓ Stores as JSON ✓ React-native API ✗ Docs are sparse ✗ Low release cadence
GrapeJS Framework-agnostic v0.22.16
BSD · Open source

Very mature, full-featured HTML/CSS page builder. Outputs raw HTML strings — not React components. Used in many no-code tools.

✓ Most battle-tested ✓ Rich plugin ecosystem ✗ Outputs HTML strings, not React ✗ Awkward in Next.js App Router ✗ Your block components can't be React
Builder.io React 19 ✓ v9.3.0
Commercial cloud service

Visual CMS and page builder. The editor lives in Builder.io's cloud — you embed content via their SDK. React 19 support is solid but you're dependent on their SaaS.

✓ Professional editor out of the box ✗ SaaS dependency, not self-hosted ✗ Pricing scales with usage ✗ Your data lives in their cloud
dnd-kit + custom React 19 ✓ v6.3.1
MIT · Bring your own everything

Use dnd-kit as the drag primitive and build the block registry, config panels, canvas, undo/redo, and JSON serialization yourself. Full control, full work.

✓ Total control over everything ✓ No abstraction leaks ✗ Building a page builder from scratch ✗ Undo/redo alone is weeks of work

Bottom line: The open-source React headless page builder space really has one serious option — Craft.js. GrapeJS is excellent but works against your React/Next.js stack. Builder.io is polished but requires their cloud. dnd-kit is the fallback if Craft.js proves problematic.
