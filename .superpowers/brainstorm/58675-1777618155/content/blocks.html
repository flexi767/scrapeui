Block System

Proposed blocks for each page type. These populate the left icon strip. Select any you'd remove or flag missing ones in the terminal.

📋 Listing Grid Page
🖼 HeroBanner

Dealer name, logo, cover photo, tagline. Full-width header.

🔍 FilterBar

Make / model / price / year filters. Toggle which filters show.

⊞ ListingGrid

The car cards. Configure columns (2–4), card style, spacing, fields shown.

⟨⟩ Pagination

Page navigation. Style: numbered / prev-next / load more.

📄 Footer

Dealer contact info, address, social links.

🚗 Listing Detail Page
🖼 ImageGallery

Photo carousel / grid. Configure layout: slider, grid, filmstrip.

💰 PriceTag

Price display with VAT toggle, currency formatting.

📊 SpecsTable

Mileage, fuel, power, transmission, year. Toggle which specs show.

📝 Description

Listing description text. Font, size, max-height configurable.

📞 CTASection

Call / email / WhatsApp buttons. Color, label, layout configurable.

🔗 RelatedListings

Similar cars strip at the bottom. Count and card style configurable.

🧱 Generic (both pages)
📦 Section

A layout container. Background, padding, width, columns. Other blocks drop inside.

T Text

Freeform heading or paragraph. Font, color, alignment.

🖼 Image

Static image or dealer logo. Width, alignment, link-on-click.

🔘 Button

Standalone CTA button. Label, color, href, size.

— Divider

Horizontal rule. Color, thickness, margin.

↕ Spacer

Empty vertical space. Height configurable.

Note: Data blocks (ListingGrid, SpecsTable, etc.) receive live data from the server at render time — dealers configure how they display, not the data itself. Generic blocks (Text, Image, Button) are fully free-form.
